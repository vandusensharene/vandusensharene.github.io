# Sharene Van Dusen — Data Analytics Portfolio

**Live site:** [vandusensharene.github.io](https://vandusensharene.github.io)

Data analytics portfolio built as part of the Lawrence Technological University / Springboard Data Analytics program. Projects demonstrate skills in Tableau, Python, SQL, Excel/Power Query, and dashboard development, grounded in 10+ years of automotive OEM operations and account management experience.

## Projects

| Project | Description | Tools |
|---|---|---|
| [Sales & Booking Ratio Dashboard](projects/sales-booking-dashboard/) | Interactive dashboard tracking revenue, bookings, and 3-month rolling booking ratios across regions and teams | HTML/JS, Chart.js, Tableau concepts |
